# 巴菲特语录HTML页面优化设计 - 羊皮纸拟物风格

## 项目概述
现有 `generate_html.py` 生成的巴菲特语录卡片HTML页面需要优化设计，用户选择了**羊皮纸拟物风格**主题。

## 设计目标
- 以羊皮纸纸张质感为基调，营造复古拟物效果
- 契合投资智慧语录的经典感和历史感
- 保持原有的搜索和标签筛选功能
- 优化阅读体验，文字清晰易读

## 主题配色方案

| 元素 | 颜色 | 说明 |
|------|------|------|
| 页面背景 | `#e8dcc4` | 浅羊皮纸底色，整体页面背景 |
| 卡片背景 | `#f4e9cb` | 略浅的羊皮纸颜色，卡片背景 |
| 卡片边框 | `#d4c49a` | 浅棕色边框，模拟纸张边缘 |
| 主文字 | `#5c431e` | 深棕褐色，羊皮纸暗字 |
| 次要文字 | `#8b6b3d` | 中棕色，来源和注释文字 |
| 分割线 | `#d4c49a` | 浅棕色，与边框同色 |

## 视觉效果设计

### 背景纹理
- 使用径向渐变模拟羊皮纸的自然不均匀质感
- 角落加深模拟自然老化效果：`radial-gradient(circle at 0 0, rgba(166, 130, 85, 0.1) 0%, transparent 50%)`

### 卡片效果
- 内阴影 `inset 0 0 30px rgba(166, 130, 85, 0.1)` 模拟纸张厚度
- 外阴影 `1px 1px 3px rgba(0,0,0,0.1)` 营造悬浮感
- 细边框 `1px solid #d4c49a` 界定卡片边界
- 圆角 `4px` 保持纸质感，不过于圆润

### 字体
- 正文使用衬线字体 `Georgia, 'Times New Roman', serif` 增强复古感
- 字间距 `letter-spacing: 0.3px` 提升可读性
- 行高 `1.7` 保持舒适阅读

### Header头部
- 保持原布局但调整配色为羊皮纸色系
- 使用棕色渐变替代原红橙色渐变

## 布局和功能保持

以下功能保持不变：
- 响应式网格布局（卡片自动排列）
- 搜索框功能（关键词搜索）
- 标签筛选功能（投资/商业/财富/人生）
- 移动端适配

## 需要修改的文件
- `generate_html.py` - 修改其中的CSS样式部分
- 重新生成 `redbook_cards.html`

## 验收标准
- 整体呈现统一的羊皮纸复古拟物风格
- 文字清晰可辨，阅读舒适
- 原有交互功能（搜索、筛选）正常工作
- 响应式布局在不同屏幕尺寸正常显示
